Pop-under Javascript

Tab under demo

This page will pop a page after loaded or on a mouse click event.

Code

<script async defer src="js/show-promote.min.js"></script>
<script>
// for tab under, you might be able to pop the ad URL directly, without worry the pop blocker.
var popURL = "//google.com/";
function onPopUnderLoaded() {
    this.frequency(1,1).useTabUnder(true).pop(popURL);
}
</script>
